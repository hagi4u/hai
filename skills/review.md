# 회고 스킬

Weekly 앱 Notion 문서를 기반으로 주간 회고를 진행하는 스킬입니다.

## 사용법

```
/회고
```

## 작동 방식

1. **Notion Weekly 데이터베이스 검색**
   - "Weekly" 데이터베이스 찾기
   - 가장 최근 주말에 생성된 문서 확인

2. **주간 활동 추출**
   - 월~금 항목에서 체크된 작업들 분석
   - 컨텍스트 스위칭 빈도 계산
   - 주요 이슈 및 완료되지 않은 항목 식별

3. **회고 템플릿 적용**
   - 다양한 도메인 활동 요약
   - 깊이 vs 넓이 분석
   - 미완료 작업 정리
   - 다음 주 개선점 제안

## 자동화된 단계

### 1단계: 데이터 수집
- Notion MCP로 Weekly 데이터베이스 접근
- 가장 최근 문서 탐색
- 월~금 컨텐츠 파싱

### 2단계: 분석
- 컨텍스트 그룹화 (도메인별)
- 완료도 평가 (체크박스 기반)
- 시간 분석 (집중 시간 추정)

### 3단계: 인사이트 도출
- "다양한 걸 많이 했지만 깊이가 부족했던" 패턴 감지
- "거의 다 왔는데 마무리 안 된" 항목 식별
- 빠른 사이클 vs 깊은 몰입 균형 제언

### 4단계: 다음 주 제안
- Finish First 전략 권장
- 컨텍스트 스위칭 최적화
- 실질적인 액션 아이템 제시

## 기대 효과

- 5분 내로 주간 회고 완료
- 자동화된 패턴 인식으로 객관적 분석
- 다음 주 개선점 구체화
- growth.md에 자동 기록 연동 가능

## 확장 가능성

- Jira/Notion 연동으로 실무 데이터 반영
- GitHub 커밋 로그와 조합
- 개인/팀 회고 모드 전환
