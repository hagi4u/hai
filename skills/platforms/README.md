# Platforms 제약사항

> TPM 역할 수행 시 참조. 프로젝트 기획/설계 단계에서 Android, iOS 제약사항을 사전에 확인한다.

## 파일 구조

| 파일 | 설명 |
|------|------|
| `android.md` | Android 플랫폼 제약사항 |
| `ios.md` | iOS 플랫폼 제약사항 |
| `biometric-webview.md` | iOS/Android 생체인식 × 웹뷰 연동 (크림페이 결제 인증) |

## 활용 방법

- 프로젝트 기획 단계에서 기능 구현 가능 여부 사전 확인
- 비개발 직군에게 제약사항을 '비용/시간'으로 환산해 전달할 때 참조
- 새로운 제약사항 발견 시 즉시 업데이트
