> 반복되는 패턴은 `context/ai_identity.md` 행동 지침에 반영한다.

## 에러 유형

| 코드 | 유형 | 설명 |
|------|------|------|
| E01 | 시간/날짜 | 요일 착각, 날짜 계산 오류 |
| E02 | 데이터 필터링 | 조건 누락, 잘못된 필터 |
| E03 | API/MCP | 연결 실패, 파싱 오류 |
| E04 | 정보 손실 | 삭제, 덮어쓰기 |
| E05 | 정보 확인 누락 | 지식베이스 미확인 후 답변 |
| E06 | 프로세스 위반 | 원칙/스킬 미준수 |
| E07 | 도구 호환성 | 플랫폼별 차이 미확인 |
